# BitPreserver

BitPreserver is a retro-inspired, nerd-approved tool to preserve and recreate floppy disk images.

Whether you're archiving your old MS-DOS floppies, installing Windows 3.1 in a VM, or exploring backups from the Amiga or Atari ST era – BitPreserver helps you read, mount, and create raw disk images from real drives or directory-based backups.

No command-line wizardry needed. Just insert, name it, click – and IMGinate your data.

**"IMGinate a future where no floppy is forgotten."** 💾✨

## Features

*   Create .img disk images from physical floppy disks
*   Mount and browse existing floppy images
*   Rebuild disk images from old file backups
*   Simple GUI built with Python and Tkinter
*   Inspired by the aesthetics of AmigaOS, Windows 3.1, and early 90s charm

## Coming Soon
